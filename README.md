# What is it?

This is a Chrome extension that removes certain HTML tags associated with YouTube Shorts.

# Install YT-NO-SHORTS extension

Firstly, you need to download the extension. Choose directory on your computer to download it and:

### 1. Cloning the repository

```bash
git clone https://github.com/Walkername/youtube-no-shorts.git
```

### 2. Enable Developer mode

Then you move to Chrome and open the toolbar. Select Extensions -> Manage Extensions.
